1.25.2ReleaseWe are excited to announce the release of Gitea 1.25.2! We strongly recommend all users upgrade to this version, as it includes important security fixes and improves overall stability. We've fixed a number of permission and authentication issues to improve security and consistency across the platform: Permission & Protection Checks Fixed permission validation for deleting releases. Corrected branch protection checks when updating pull requests via rebase. Fixed permission checks for issue dependencies. Fixed validation for deleting comment history entries.
