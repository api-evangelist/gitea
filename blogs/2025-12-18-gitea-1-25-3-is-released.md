1.25.3ReleaseWe’re excited to announce the release of Gitea 1.25.3! We strongly recommend all users upgrade to this version, as it includes important security fixes, numerous bug fixes, and overall stability improvements. In this release, Gitea has been upgraded to Go 1.25.5, which addresses two security vulnerabilities in crypto/x509: GO-2025-4175 and GO-2025-4155. We would like to thank @silverwind for providing the patches. This release includes 23 merged pull requests, thanks to the amazing contributions from our community.
