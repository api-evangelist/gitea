1.24.7ReleaseWe are excited to announce the release of Gitea version 1.24.7! We strongly recommend all users upgrade to this version for improved stability and security. This release includes 7 merged pull requests, thanks to the amazing contributions from our community. Security​ This release addresses several important security vulnerabilities: LFS Authentication Bypass A user without valid credentials could upload or download LFS files by submitting a malformed JWT token. Thanks to Scott Tolley from Black Duck for reporting this issue, and to @wxiaoguang for the fix in https://github.
