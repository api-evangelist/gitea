1.24.2ReleaseWe are excited to announce the release of Gitea version 1.24.2! This is a quick bug fix release that resolves a Docker push issue present in v1.24.1. A total of 3 pull requests have been merged into this release, thanks to the continued contributions from our community. How to install or update​ Download our pre-built binaries from the Gitea downloads page — make sure to select the version compatible with your platform.
