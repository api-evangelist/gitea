We are happy to announce the release of Gitea Runner 1.0.0. This is the first stable release of the official Gitea Actions runner under its new name. The project formerly known as act_runner has been renamed to Gitea Runner, with the repository now available at gitea/runner. Why the rename?​ The original act_runner name reflected the project's early implementation history and its relationship with nektos/act. Since then, Gitea Actions and the runner have grown into a core part of the Gitea ecosystem. The new name makes the purpose clearer: this is the official runner for Gitea Actions.
