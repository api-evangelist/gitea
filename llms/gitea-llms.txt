# Gitea

> Gitea is an open-source, self-hosted Git service and all-in-one DevOps platform written in Go (MIT licensed). It offers Git hosting, code review, issue tracking, project boards, a 24+ format package registry, and GitHub-Actions-compatible CI/CD (Gitea Actions). Every instance ships a Swagger 2.0 REST API at /api/v1/ (~300 operations). CommitGo, Inc. offers Gitea Enterprise and single-tenant Gitea Cloud.

## APIs

- [Gitea REST API](https://docs.gitea.com/api/): ~300-operation Swagger 2.0 API at /api/v1 (repos, users, orgs, issues, PRs, packages, notifications, admin).
- [Gitea Actions API](https://docs.gitea.com/usage/actions/overview): GitHub-Actions-compatible workflow/runs/jobs/runners/secrets under /repos/{owner}/{repo}/actions/*.
- [Gitea Package Registry](https://docs.gitea.com/usage/packages/overview): multi-format registry (Cargo, Container/OCI, Maven, npm, NuGet, PyPI, Helm, ...) under /api/packages/{owner}/{type}.
- [Gitea Webhooks](https://docs.gitea.com/usage/webhooks): outbound JSON event delivery for push/PR/issue/release/package events.
- [Gitea Cloud Management](https://about.gitea.com/products/cloud): managed single-tenant instance provisioning console.
